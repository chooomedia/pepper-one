== Description ==

Simple dark design template based on Twitter Bootstrap (the most popular HTML, CSS and JS framework for developing fast-reacting, mobile first projects on the web) and Font Awesome (or Fontawesome, a complete suite of over 675 pictographic symbols for easily scalable vector graphics on websites). This theme, like WordPress, is licensed under the GPL. You can use this design as a basis for your next WordPress theme project and you can remove the link in the footer or do whatever you want. Your feedback on how we can continuously improve the Pepper-One template is highly appreciated. The page templates include the right sidebar (default page template), the left sidebar, the full width, the space with container, the space without container page. Other features - Currently available with Bootstrap v4.0.0.0-beta.2, Widgetized footer, WooCommerce ready, compatible with Contact Form 7, compatible with Visual Composer, compatible with Elementor Page Builder. This topic will be an active project that we will update from time to time. Check this page regularly for updates.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
